# Project Summary

## GitHub Repository Description

Simulation-first Microsoft 365 identity lifecycle automation lab using Python, PowerShell, mock Microsoft Graph workflows, configuration-driven access mapping, reports, and CI.

## Short LinkedIn Featured Description

Simulation-first Microsoft 365 identity lifecycle automation lab showing onboarding and offboarding workflow planning with Python, PowerShell, mock Microsoft Graph actions, JSON configuration, reports, and CI.

## Technical Summary

This project demonstrates how identity lifecycle actions can be validated, planned, reviewed, and reported without connecting to a real tenant or executing production changes.

## What It Demonstrates

- Configuration-driven group, license, and SaaS access mapping.
- Mock Microsoft Graph action planning for onboarding and offboarding.
- Public-safe reporting, tests, and documentation for a simulation-first automation lab.
