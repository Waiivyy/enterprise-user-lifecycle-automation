# LinkedIn Featured Description

Enterprise User Lifecycle Automation is a public-safe portfolio demo for Microsoft 365 identity lifecycle automation.

It shows how onboarding and offboarding workflows can be planned with Python, PowerShell, mock Microsoft Graph actions, JSON configuration, SaaS provisioning payloads, Markdown/JSON reports, and CI checks.

The project is intentionally simulation-first: it uses `example.invalid`, fake group IDs, mock Graph responses, and production execution is disabled. It demonstrates automation design and security boundaries without connecting to a real tenant.
